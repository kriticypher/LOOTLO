LOOTLO - Minimalist E-Commerce Platform

LOOTLO is a clean, responsive, and lightweight e-commerce web application built using
React. Designed with minimalism in mind, the platform focuses on a seamless user
experience, providing a high-fidelity storefront demonstration without the complexity of a
backend.

The project is a minimalistic static website that utilises a structured product.json file to
mimic a live database, ensuring fast performance and a smooth browsing experience.

🚀 Features

• Product Categorisation: Browse products through organised categories for an
intuitive shopping experience.

• Working Search Bar: A functional search component that filters through the
product library instantly.

• Fully Responsive Design: Optimised for a seamless experience across devices,
ensuring the platform is fully accessible on both laptops and mobile phones.
• Interactive UI:

◦ Cart & Wishlist Functionality: Integrated interfaces to manage your
selections and save favourites.

◦ Real-time Notifications: Instant feedback via React Toastify for a modern
feel.

• Minimalist Mock Backend: Utilises a sample product.json file to simulate real-
world data flow within a single-page architecture.

🛠 Tech Stack

• Frontend Framework: React.js (Functional components with Hooks)
• Build Tool & Dev Server: Vite (for optimised development and bundling)
• Styling: Tailwind CSS (utilising utility classes and custom hex palettes.
• Icons: Lucide React (providing clean UI icons for Heart, Cart, and Search)
• Routing: React Router DOM (handling seamless navigation throughout the app)
• State Management: React Context API (managing global application states)
• Notifications: React Toastify (for interactive pop-up messages)

🏁 Getting Started

Prerequisites
To run this project locally, you must have Node.js and npm (Node Package Manager)
installed on your system. You can download them from nodejs.org.

Installation
Once npm is installed, follow these steps:

1. Clone the repo
git clone https://github.com/yourusername/lootlo.git

2. Navigate to the project folder
cd lootlo

3. Install dependencies (This uses npm to download the required libraries)
npm install

4. Run the application
npm run dev
